# ODEs_dataset

Dataset and benchmark scaffolding for ordinary differential equation (ODE) systems.

## Layout

- `docs/spec/`: dataset specification and registries.
- `configs/`: declarative configuration files for systems, observations, splits, tasks, benchmarks, and releases.
- `src/`: Julia source modules for generation, dataset handling, evaluation, registries, and utilities.
- `data/`: raw data, processed data, manifests, and release artifacts.
- `experiments/`: smoke tests and baseline experiment workspaces.
- `reports/`: generated tables, plots, and logs.
- `test/`: unit, integration, and regression tests.
